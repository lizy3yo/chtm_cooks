# Constant Items Feature - Quick Guide

## For Custodians

### How to Mark Items as Constant

1. **Navigate to Inventory Management**
   - Click "Inventory" in the sidebar

2. **Create or Edit an Item**
   - Click "Add New Item" or edit existing item
   - Fill in item details

3. **Enable Constant Status**
   - Check the box: ☑ "Mark as Constant Item"
   - Description explains: "Constant items always appear on student request forms"

4. **Save the Item**
   - Click "Add Item" or "Update Item"
   - Item now appears in "Constant Items" tab

### Managing Constant Items

**View All Constant Items:**
- Go to Inventory Management
- Click the "Constant" tab (emerald colored)
- See all items marked as constant

**Remove Constant Status:**
- Option 1: Click "Remove" button in Constant Items tab
- Option 2: Edit item and uncheck the constant checkbox
- Option 3: Click item → Menu (⋮) → "Remove Constant"

**Visual Indicators:**
- Emerald "Constant" badge on items
- Dedicated tab with count
- Clear menu options

## For Students

### Using Constant Items on Request Form

1. **Open Request Form**
   - Navigate to "Request Equipment"
   - See "Frequently Requested Equipment" section at top

2. **Browse Constant Items**
   - View all items marked as constant by custodians
   - See availability status for each item
   - Items displayed in attractive grid layout

3. **Add Items to Request**
   - Click any available item card
   - Item automatically added to "Selected Items"
   - Adjust quantity as needed

4. **Visual Feedback**
   - ✓ Green badge: "X available" - Can request
   - ⚠ Amber badge: "Out of stock" - Cannot request
   - "Added" badge: Already in your cart

5. **Complete Request**
   - Fill in borrow period, purpose, etc.
   - Submit request as normal
   - Constant items included in submission

## Tips & Best Practices

### For Custodians

**What to Mark as Constant:**
✓ Frequently requested items
✓ Essential lab equipment
✓ Items students always need
✓ High-demand tools

**What NOT to Mark as Constant:**
✗ Rarely used specialty items
✗ Items with very limited quantity
✗ Seasonal equipment (unless in season)
✗ Items under maintenance

**Maintenance:**
- Review constant items monthly
- Remove items no longer frequently needed
- Add new popular items
- Keep list relevant and useful

### For Students

**Best Practices:**
✓ Check constant items first
✓ Verify availability before requesting
✓ Request only what you need
✓ Return items on time for others

**If Item Not Available:**
- Check "Add Item" for alternatives
- Browse full catalog
- Contact custodian for assistance
- Plan ahead for future requests

## Troubleshooting

### Custodian Issues

**Q: Item marked as constant but not showing in tab?**
A: Refresh the page. Check if item is archived.

**Q: Can't toggle constant status?**
A: Ensure you have custodian permissions. Check browser console for errors.

**Q: How many items can be constant?**
A: No limit, but recommend 10-20 for best UX.

### Student Issues

**Q: Not seeing constant items section?**
A: No items marked as constant yet. Contact custodian.

**Q: Can't click constant item?**
A: Item may be out of stock or already in your cart.

**Q: Constant item disappeared?**
A: Custodian may have removed constant status. Check regular catalog.
