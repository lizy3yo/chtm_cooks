Student Responsibilities

  • Browse Catalog: View available inventory items and their details
  • Submit Requests: Request items from the inventory for personal or academic use
  • Track Requests: Monitor the status of submitted requests (pending, approved, rejected)
  • Report Issues: Report damaged, lost, or defective items to custodians
  • View Item Status: Check item condition and availability before requesting
  • Limited Access: Read-only access to catalog, write access only for requests and issue reports

Key Processes

  • 1.0 Authentication & Authorization: Validates student credentials and grants appropriate access
  • 3.0 View Catalog: Displays available inventory items with details and availability status
  • 4.0 Submit & Track Requests: Handles item request submission and status tracking
  • 6.0 Report Issues: Allows students to report item damage, defects, or other issues

Student Workflow

  • Step 1: Login to the system with student credentials
  • Step 2: Browse the catalog to view available items
  • Step 3: Submit a request for needed items
  • Step 4: Track request status (pending approval from custodian)
  • Step 5: Report any issues with borrowed items
