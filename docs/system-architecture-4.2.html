4.2 Proposed System Architecture

Industry-standard high-level architecture showing end-to-end connectivity across Web and Mobile channels, Cloud Server services, data stores, resilience layer, and AI chatbot integration.

Architecture Style: Cloud-native modular monolith Topology: Primary Region + Secondary Region (DR) Interfaces: HTTPS REST APIs + event-driven processing AI Layer: Chat orchestrator + external LLM provider
