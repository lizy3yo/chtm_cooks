Process Descriptions

  • 1.0 Authentication & Authorization: Validates user credentials and manages access control for all user roles
  • 2.0 Manage Users: Handles user account creation, updates, and student information management
  • 3.0 Manage Inventory: Controls inventory items including adding, updating, archiving, and tracking stock levels
  • 4.0 Process Requests: Manages student requests for items, approval workflows, and request fulfillment
  • 5.0 Manage Donations: Records and tracks donated items, integrates with inventory system
  • 6.0 Item Inspection: Handles item condition checks, damage reports, and quality assessments
  • 7.0 Generate Reports: Creates system-wide reports, analytics, and inventory summaries for different user roles

User Role Descriptions

  • Superadmin: Full system access, manages users, views all reports, system configuration
  • Instructor: Monitors inventory, views student data, accesses analytics and donation records
  • Custodian: Manages inventory, processes requests, records donations, conducts inspections
  • Student: Submits item requests, views request status, reports item damage or issues
